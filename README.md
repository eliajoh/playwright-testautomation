# Playwright Test Automation Project

A **TypeScript**-based test automation suite using [Playwright](https://playwright.dev/) for end-to-end testing. This project includes both **positive and negative test cases**. This project is made for learning purposes.

---

## Features
- **End-to-end testing** with Playwright
- **Positive and negative test cases** for comprehensive coverage
- **Coming soon: CI/CD** with Docker and Jenkins


---

## Test Strategy

The test strategy for this project was designed to ensure efficient and maintainable test coverage, focusing on important scenarios and basic regression-tests while mitigating the risk of test inflation. Test cases were selected based on a combination of domain expertise and risk assessment, ensuring that only meaningful and impactful tests are implemented. This approach guarantees a scalable and sustainable test suite, aligned with the evolving needs of the website under test.
